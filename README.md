# mink-frontend-sb

Språkbanken instance code for the [Mink frontend](https://github.com/spraakbanken/mink/).

To develop the Språkbanken instance, clone this repo next to the core repo and create a symlink from `mink-frontend/instance` to `mink-frontend-sb`.

To develop your own instance, first see [DEVELOPMENT.md](https://github.com/spraakbanken/mink-frontend/blob/main/docs/DEVELOPMENT.md) of the core repo.
Then, take a look at our [plugin.ts](./plugin.ts) for hints on what you can do.
